# h5-dray

> 基于h5开发的拖拽组件

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

``` 

如果不是很了解h5拖拽api的一些特性，请移步->
